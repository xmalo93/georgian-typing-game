# ქართული ტაიპინგ თამაში (Ball Jump Typing Game)

ეს არის პატარა, ფერადი, ბავშვებისთვის მეგობრული ვებ-თამაში:
- ეკრანზე გამოდის ქართული სიტყვა
- მოთამაშე აკრეფს სიტყვას და დააჭერს Enter-ს
- სწორად: +50 ქულა და ბურთი ახტება ზემოთ
- შეცდომაზე: −10 ქულა (თამაში არ სრულდება)
- ყოველ 500 სწორად სიტყვაზე იცვლება გარემო: ეზო → ღრუბლები → კოსმოსი
- სიჩქარის სლაიდერი არეგულირებს რამდენ წამში იცვლება სიტყვა ავტომატურად

## ფაილები
პროექტში არის 4 ფაილი:
- `index.html`
- `style.css`
- `script.js`
- `README.md`

## როგორ გავუშვა ლოკალურად (კომპიუტერზე)
1. ჩამოტვირთე/დააკოპირე პროექტის ფაილები ერთ საქაღალდეში
2. უბრალოდ გახსენი `index.html` ბრაუზერში (Chrome/Edge/Firefox)

## GitHub Pages-ზე ატვირთვა და გაშვება
### 1) GitHub-ზე ახალი Repository
1. შედი GitHub-ზე
2. დააჭირე **New repository**
3. დაარქვი მაგალითად: `georgian-typing-game`
4. მონიშნე **Public** (ან Private თუ გინდა, მაგრამ Pages ზოგჯერ Public-ზე მარტივია)
5. შექმენი Repo

### 2) ფაილების ატვირთვა
Repo-ში:
1. დააჭირე **Add file → Upload files**
2. ატვირთე ეს 4 ფაილი: `index.html`, `style.css`, `script.js`, `README.md`
3. დააჭირე **Commit changes**

### 3) GitHub Pages ჩართვა
1. Repo-ში შედი **Settings**
2. მარცხნივ აირჩიე **Pages**
3. **Build and deployment** სექციაში:
   - **Source**: აირჩიე `Deploy from a branch`
   - **Branch**: აირჩიე `main` და `/ (root)`
4. დააჭირე **Save**
5. რამდენიმე წამში/წუთში GitHub გაჩვენებს ლინკს:
   - `https://<username>.github.io/georgian-typing-game/`

## შენიშვნები
- თამაში მუშაობს სუფთა Vanilla JavaScript-ით (framework-ების გარეშე)
- სიტყვების სია არის `script.js`-ში მასივში `WORDS`
- თუ გინდა მეტი სიტყვა, უბრალოდ დაამატე `WORDS`-ში
