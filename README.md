# ქართული ტაიპინგ თამაში (Ball Jump Typing Game)

ეს არის პატარა, ფერადი, ბავშვებისთვის მეგობრული ვებ-თამაში:
- ეკრანზე გამოდის ქართული სიტყვა
- მოთამაშე აკრეფს სიტყვას და დააჭერს Enter-ს
- სწორად: +50 ქულა და ბურთი ახტება ზემოთ
- შეცდომაზე: −10 ქულა (თამაში არ სრულდება)
- ყოველ 500 სწორად სიტყვაზე იცვლება გარემო: ეზო → ღრუბლები → კოსმოსი
- სიჩქარის სლაიდერი არეგულირებს რამდენ წამში იცვლება სიტყვა ავტომატურად

## ფაილები
პროექტში არის 4 ფაილი:
- `index.html`
- `style.css`
- `script.js`
- `README.md`

## შენიშვნები
- თამაში მუშაობს სუფთა Vanilla JavaScript-ით (framework-ების გარეშე)
- სიტყვების სია არის `script.js`-ში მასივში `WORDS`
- თუ გინდა მეტი სიტყვა, უბრალოდ დაამატე `WORDS`-ში
